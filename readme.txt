== Description ==

### The Easiest way to create responsive, interactive and animated charts in Wordpress.

Select a Chart Block from the availabe chart types. Currently there are 6 different Chart Blocks: 

1. Bar Chart Block
2. Line Chart Block
3. Pie Chart Block
4. Doughnut Chart Block
5. Horizontal Bar Chart Block
6. Horizontal Line Chart Block

Upload your CSV file and TADAA! 

Your Beautiful, Animated and Responsive chart is ready for the world!

== Installation ==

1. Log in to your WordPress dashboard
2. Visit Plugins > Add New
3. Search for Charts Blocks
4. Install and Activate

== Frequently Asked Questions ==

= Is Charts Blocks Free =

Yes, Charts Blocks is 100% Free.

= How do I create my first Chart =

Navigate to the Page or Post where you want to add a chart. Select one Chart Block from the available chart blocks. Upload your CSV file. Your beautiful chart is ready for the world.

= Why is my CSV data not showing in Charts =

Download the example CSV file provided below the Upload CSV button. Make sure your data is in the specified format.

= How can I add Grouped Bar Charts or Multiline Charts =

Grouped Bar Charts, Multiline Charts and many more Chart Types will be added in the upcoming versions.
